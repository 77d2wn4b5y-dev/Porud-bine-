# Porudžbine

Jednostavna mobilna aplikacija za trgovačkog putnika. Radi u pregledaču i čuva podatke lokalno na uređaju.

## Funkcije

- unos ili izbor objekta
- automatski datum i vreme
- kolona **Prethodni put**
- uvek prazna kolona **Danas** za novu porudžbinu
- numerička tastatura na telefonu i iPadu
- automatski prelazak na sledeći artikal pritiskom na Enter/Next
- napomena za objekat
- veliko dugme za čuvanje, stalno vidljivo pri dnu
- istorija porudžbina u istom prozoru
- kopiranje, ponovno učitavanje i brisanje porudžbine
- lokalno čuvanje u `localStorage`

## Pokretanje

Otvori `index.html` u pregledaču ili uključi GitHub Pages za granu `main` i folder `/root`.

## Važno

Podaci se čuvaju samo u pregledaču na uređaju na kojem se aplikacija koristi. Brisanje podataka pregledača briše i porudžbine. Rezervna kopija i sinhronizacija mogu se dodati u sledećoj verziji.
